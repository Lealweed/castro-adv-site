# CRM Jurídico SaaS — Frontend (FASE 2)

## Stack
- React + Vite + TypeScript
- Tailwind CSS
- React Router
- TanStack Query (base)

## Rodar local
```bash
cd crm-juridico-frontend
npm install
npm run dev
```

## Páginas
- / Dashboard
- /clientes
- /clientes/:clientId
- /financeiro
- /relatorios-ia
- /portal
- /configuracoes

Tudo mockado por enquanto (sem backend).
